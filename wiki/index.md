# Проектирование платформы управления учебными проектами

Доменные области
- Распределение по командам
- Управления проектами

### Функции:

#### Лендинг
На Лендинге будут отображаться публичные проекты


#### Роли Пользователей:
Участник, Руководитель, Админ, СуперАдмин

У каждой роли, есть свой перемишены на чтение запись и редактирования, 
но так же у пользователя могу быть пользовательские перемишены.

Пользовательские перемишены имеют больший приоритет, чем пермишен 
для Роли.

#### Пользователи:
- Аудит лог все действия пользователя на платформе (Отображение, как в Github)

#### Уведомление:
- Почта/Tg_bot

#### Резюме
- CRUD + импорт/экспорт 
- Поиск по навыкам.

- Отправить приглашение в команду (Руководитель) (Уведомление)

#### Отклики
- CRUD + импорт/экспорт

- Откликнуться на проект (Участник) (Уведомление)
- Отправить приглашение в команду (Руководитель) (Уведомление)
- Принять приглашение (Участник) (Уведомление)

#### Проекты
- CRUD + импорт/экспорт
- Список участников 
- Список откликов
