### Функции:

#### Лендинг
На Лендинге будут отображаться публичные проекты
Краткая информация об платформу возможность её преимуществах 


#### Роли Пользователей:
Участник, Руководитель, Админ, СуперАдмин

У каждой роли, есть свой перемишены на чтение запись и редактирования, 
но так же у пользователя могу быть пользовательские перемишены.

Пользовательские перемишены имеют больший приоритет, чем пермишен 
для Роли.

#### Пользователи:
- Аудит лог все действия пользователя на платформе (Отображение, как в Github)

#### Уведомление:
- Почта/Tg_bot

#### Резюме
- CRUD + импорт/экспорт 
- Поиск по навыкам.

- Отправить приглашение в команду (Руководитель) (Уведомление)

#### Отклики
- CRUD + импорт/экспорт

- Откликнуться на проект (Участник) (Уведомление)
- Отправить приглашение в команду (Руководитель) (Уведомление)
- Принять приглашение (Участник) (Уведомление)

- Хм.. Возможно необходима сущность вакансии (Это усложнение, но позволит для Lisa подтверждать найм)
- Либо будет ещё один статус перед созданием утверждение вакансии. Нужно уточнять у tg@dmitriy_a_fedorov

#### Проекты
- CRUD + импорт/экспорт
- Список участников 
- Список откликов
- Статус проекта зависит от Пространства

- Добавление (CRUD) Участника в команду  

* Настройки
    * Подчиняются настройкам в рамках, которого Пространство создан проект
    * Публичный проект 
    * Настройка максимального количества участник для проекта (Подстраивается под количество участников пространства)
